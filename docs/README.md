# UTask
Pet project for task tracking via telegram bot

Требования к проекту
------------------------------

Через интерфейс телеграм бота можно:
1. Получить список участников
2. Получить список групп
3. Получить список участников в группе
4. Получить список проектов
5. Получить список участников в проекте
6. Добавить участника в проект 
7. Удалить участника из проекта
8. Добавить группу в проект
9. Удалить группу из проекта
10. Получить список всех заданий (с фильтрами открыта/выполнена/в работе/закрыта/тестируется)
11. Получить список всех заданий проекта (с фильтрами открыта/выполнена/в работе/закрыта/тестируется)
12. Получить список всех заданий участника (с фильтрами открыта/выполнена/в работе/закрыта/тестируется)
13. Добавление задания участнику
14. Изменение статуса задачи
15. Просмотр материалов по задаче

Пока аутотенфикация через телеграм, дополнительной не требуется
